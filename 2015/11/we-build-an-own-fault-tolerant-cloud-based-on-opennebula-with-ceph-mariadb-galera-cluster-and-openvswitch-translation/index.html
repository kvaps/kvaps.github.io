We build an own fault-tolerant cloud based on OpenNebula with Ceph, MariaDB Galera Cluster and OpenvSwitch [Translation]

November 16, 2015

This time I would like to tell how to configure this subject, in a particular each separate component as a result to receive the own, expanded, otkazoustoycheavy cloud based on OpenNebula. In this article I will consider the next moments:

  • The Ceph installation, the distributed storage. (I will describe installation of two-level storage with the caching pool from SSD disks)
  • The installation MySQL, Galera of a cluster with the master master replication
  • Installation software switch of OpenvSwitch
  • Installation directly OpenNebula
  • Setup of failover cluster
  • Initial configuration

Subjects in itself very interesting so even if you are not interested in an ultimate goal, but setup of some separate component interests. I ask favor under cut.
